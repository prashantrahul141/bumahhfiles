Uploading

Files can be uploaded via curl using the following:
    curl -F file=@"./miku.png" {{ domain }}


even multiple files in a single request using:
    curl -F file=@"./miku.png" -F file=@"./teto.png" {{ domain }}


Or you can use web upload option




Deleting
With every upload you get a del_key. Which you can use to delete your files by either opening: 

    {{ domain }}/d/{filename}?del_key={del_key}

 example:
    {{ domain }}/d/0eHak-miku.png?del_key=upZ9s

in your browser

Or by sending a DELETE request to:

    {{ domain }}/{filename}?del_key={del_key}

example:
    curl -X DELETE {{ domain }}/0eHak-miku.png?del_key=upZ9s
